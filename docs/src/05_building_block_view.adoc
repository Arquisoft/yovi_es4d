== Vista de Bloques
La vista de bloques muestra el comportamiento interno de la aplicación haciendo uso de diagramas de componentes que ilustran el funcionamiento de los microservicios que emplea la aplicación. 


=== Representación del sistema general
image::diagrama_componentes_vista_bloques_yovi_V0_1.png[]
Motivación::
Yovi permite interactuar con el sistema, hace que los usuarios se puedan registrar y hacer uso del juego. Por otro lado, de manera interna, tenemos los microservicios los cuales interactuan con la aplicación web. Finalmente, el microservicio de usuarios se conecta con la base de datos para persistir la información.


.Descripción general del sistema
[options="header",cols="1,2"]
|===
| Actores | Descripción
| Usuario | Cliente/Usuario de la aplicación que interactúa con Yovi.
| Yovi | Sitema que emplean los usuarios para poder hacer uso del juego.
| WebApp | Contiene la interfaz de usuario de la aplicación.
| Gateway | Api que permite conectar todos los microservicios en un único paquete.
| UserService | Gestiona la creación de nuevos usuarios e interactua con la base de datos.
| Base de Datos | Base de datos donde se encuentran los usuarios y las estadísticas de las partidas.
|===
